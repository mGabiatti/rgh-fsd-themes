# FSD Themes Vault

> A personal collection of themes for **FreeStyle Dash (FSD)** on Xbox 360 RGH/JTAG.

---

## 🚀 How to Install a Theme on FSD

1. Connect your Xbox to your PC via **FTP** (IP shown in FSD main screen)
2. Navigate to: `HDD:\Freestyle\Themes\`
3. Create a folder with the theme name and place the `.xzp` file inside it
4. In FSD: **Settings → Appearance → Theme** → select the theme
5. Restart FSD if needed

> 💡 Recommended FTP tools: **FileZilla** or **Total Commander**
